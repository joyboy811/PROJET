\section{Introduction}
Dans le cadre du développement de l’application PAGe Platform, une combinaison de technologies modernes est essentielle pour créer une
solution performante et efficace. Chaque outil utilisé joue un rôle crucial dans la structuration, la présentation et l’interactivité de l’application, permettant ainsi une gestion optimisée des risques algorithmiques. Ce section présente les principaux outils et technologies employés dans le développement de cette application, en mettant en avant leurs fonctionnalités et leurs avantages.
\section{Outils et technologies utilisés}
\subsection{Introduction}
\subsection{VS Code}
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/VSCode logo.png}
    \caption{Logo de VS Code}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}Visual Studio Code (VS Code) est un environnement de développement intégré (IDE) léger et performant développé par Microsoft. Il offre de nombreuses fonctionnalités telles que la coloration syntaxique, l’autocomplétion du code, le débogage intégré et la gestion des extensions.
\subsection{React}
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/React logo.png}
    \caption{Logo de React}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}React est une bibliothèque JavaScript open source développée par Meta pour la création d’interfaces utilisateur interactives et dynamiques. Basée sur le concept des composants réutilisables, elle facilite le développement d’applications web modernes à page unique (Single Page Applications - SPA).
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/Django logo.png}
    \caption{Logo de Django}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}Django est un framework web open source écrit en Python qui suit l’architecture MVT (Model-View-Template). Il fournit de nombreux outils intégrés pour le développement rapide et sécurisé d’applications web, notamment la gestion des utilisateurs, l’administration et l’accès aux bases de données.
\subsection{PostgreSQL}
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/PostgreSQL Logo.png}
    \caption{Logo de PostgreSQL}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}PostgreSQL est un système de gestion de base de données relationnelle et objet-relationnelle open source reconnu pour sa robustesse, sa fiabilité et ses performances. Il permet le stockage, la gestion et la manipulation sécurisée des données.
\subsection{Docker}
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/Docker logo.png}
    \caption{Logo de Docker}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}Docker est une plateforme de conteneurisation permettant d’empaqueter une application ainsi que toutes ses dépendances dans des conteneurs portables et reproductibles. Cette technologie facilite le déploiement et garantit un environnement d’exécution identique sur toutes les machines.
\subsection{Postman}
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/Postman logo.png}
    \caption{Logo de Postman}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}Postman est un outil de développement d’API permettant de tester, documenter et déboguer les services web. Il offre une interface intuitive pour envoyer des requêtes HTTP et analyser les réponses obtenues. Dans notre projet, Postman a été utilisé pour vérifier le bon fonctionnement des API développées avec Django, tester les opérations CRUD et valider les échanges de données entre le frontend et le backend.
\subsection{Git}
\begin{figure}[h!]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/git logo.jpg}
    \caption{Logo de Git}
    \label{fig:exempledefigure}
\end{figure}
\paragraph{}Git est un système de gestion de versions distribué permettant de suivre les modifications du code source et de faciliter le travail collaboratif entre plusieurs développeurs. Grâce à ses fonctionnalités de branches, de fusion et d’historique des modifications, Git assure une meilleure organisation du développement logiciel.
\subsection{Github}
\vspace{2.9cm}
\begin{figure}[h]
    \centering
    \includegraphics[width=0.3\textwidth]{Logos/Github logo.png}
    \caption{Logo de Github}
    \label{fig:exempledefigure}
\end{figure}

\paragraph{}GitHub est une plateforme d’hébergement de dépôts Git qui facilite la collaboration entre développeurs. Elle offre des fonctionnalités avancées telles que la gestion des branches, les Pull Requests, le suivi des problèmes (Issues) et l’intégration continue. Dans notre projet, GitHub a été utilisé pour héberger le code source, centraliser les contributions de l’équipe et assurer le suivi de l’évolution du projet tout au long de son développement.
\subsection{Conclusion}
Pour le développement de notre projet, le choix des technologies et outils
utilisés s’est fait en tenant compte à la fois de la robustesse, de la facilité de développement et de la maintenabilité de l’application tout en assurant sa qualité, sa scalabilité et sa sécurité.
